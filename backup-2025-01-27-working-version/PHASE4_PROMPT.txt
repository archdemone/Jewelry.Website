Let's proceed with Phase 4: Home Page Development. This phase will create a stunning, conversion-optimized home page for the jewelry store.

## Phase 4 Tasks:

### 1. Create Home Page Components Structure
Create the following folder structure:
/components/
  /home/
    - HeroSection.tsx
    - FeaturedProducts.tsx
    - CategoryShowcase.tsx
    - TestimonialsSection.tsx
    - NewsletterSection.tsx
    - WhyChooseUs.tsx
    - InstagramFeed.tsx (optional)
    - BrandsSection.tsx (optional)

### 2. Install Additional Required Components
npx shadcn-ui@latest add carousel
npx shadcn-ui@latest add card
npx shadcn-ui@latest add aspect-ratio
npm install embla-carousel-react embla-carousel-autoplay

### 3. Create Hero Section Component
Create /components/home/HeroSection.tsx

Features:
- Full-width hero with 70vh height
- Image carousel/slider with 3-4 high-quality jewelry images
- Autoplay with pause on hover
- Overlay with text content:
  - Headline: "Exquisite Jewelry for Every Moment"
  - Subheadline: "Handcrafted with Love, Designed for Life"
  - Two CTA buttons: "Shop Collection" and "View Catalog"
- Smooth transitions between slides
- Mobile responsive with adjusted height and text sizing
- Optional: Video background option

Example Structure:
const slides = [
  {
    image: "/images/hero/hero-1.jpg",
    title: "New Summer Collection",
    subtitle: "Discover timeless elegance",
    cta: "Shop Now",
    link: "/products"
  },
]

### 4. Create Featured Products Section
Create /components/home/FeaturedProducts.tsx

Features:
- Section title: "Featured Collection"
- Grid of 4-6 featured products from database
- Product cards with:
  - Image with hover effect (zoom or secondary image)
  - Product name
  - Price (with sale price if applicable)
  - "Quick View" button on hover
  - "Add to Cart" button
  - Wishlist heart icon
- Link to "View All Products"
- Fetch actual products from database marked as featured

Database Query:
const featuredProducts = await prisma.product.findMany({
  where: { featured: true, active: true },
  take: 6,
  include: { category: true }
})
